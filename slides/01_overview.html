Portability

C and C++ are one of the most portable languages

  • there are compilers for every platform
  • the 3 most popular compilers cover all desktop platforms, iOS, Android, PlayStation 4, XBox
This may sound strange. As long as you are using standard C++ you have covered more platforms than with any other language. For example if you have a library that handles the platform specific window creation, drawing and reading input like SDL2.0, you get all PC platforms, iOS and Android.
